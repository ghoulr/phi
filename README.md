# phi monorepo

Monorepo for `phi` and installable `pi` extensions.

## Layout

```text
packages/
笏懌楳 phi/      # main application
笏披楳 pi-*/     # installable pi extensions
```

## Commands

- `bun run tui`
- `bun run dev:service`
- `bun run check`
- `bun run test:agent`
